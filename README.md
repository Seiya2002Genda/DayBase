# DayBase_ver_02: NPBスコアボードシミュレータ (ルールベース) ⚾

## 概要

`DayBase_ver_02` は、Pythonで実装されたNPB（日本野球機構）の試合をシミュレーションするためのコンソールアプリケーションです。従来のR, H, Eのスコア手入力方式から進化し、**ボール (B), ストライク (S), アウト (O)** のカウントと、**二塁打 (2B), 三塁打 (3B), 本塁打 (HR)** を含むルールベースのシミュレーションを採用しています。

ユーザーは、チーム、ラインナップ、先発投手を設定し、一球ごとの入力を通して試合を進行させることができます。最終スコア、安打、失策、および詳細な打席記録はCSVファイルに保存されます。

---

## 機能ハイライト

* **NPBリーグ選択:** セントラル・リーグとパシフィック・リーグからチームを選択。
* **ラインナップ設定:** ホームチームとアウェイチームのスターティングメンバー9人を手動で設定。
* **先発投手設定:** 各チームの先発投手を設定し、対戦カードに表示。
* **ルールベースシミュレーション:**
    * **4ボール、3ストライク、3アウトの原則**を厳守。
    * 打席結果として、**1B, 2B, 3B, HR, Walk, Error, Out** を入力可能。
    * 安打種別に応じた**走塁処理**と得点計算。
* **スコアボード表示:** 各イニング終了時にR, H, Eを含むスコアボードをコンソールに表示。
* **データ保存:** 試合終了後、詳細なスコア情報（R, H, E）をCSVファイルに自動保存。

## 📁 ファイル構成

このプロジェクトは、機能ごとにファイルを分割して構成されています。

| ファイル名 | 役割 |
| :--- | :--- |
| `main.py` | プログラムのエントリポイント。全クラスを呼び出し、試合のメインループ（イニング進行）を管理する。 |
| `NipponProfessionalBaseball.py` | NPBのリーグ定義と、スコアボードの描画ロジックを格納する。 |
| `SetVersus.py` | リーグおよび対戦チーム（ホーム/アウェイ）の選択ロジックを格納する。 |
| `Player.py` | 選手オブジェクトの定義と、チームのラインナップ（打順）管理を行う。 |
| `Versus.py` | 投手と打者の対戦カード (`VersusCard`) を表示するためのクラスを定義する。 |
| `Inning.py` | **野球の基本ルール**に基づき、一打席ごとのカウント、走塁、アウト処理、およびイニング内のスコア計算（R, H, E）を担当する。 |
| `GameScore.py` | 試合全体（9イニング）のスコア（R, H, E）および全打席の履歴を記録・集計する。 |
| `SaveCSV.py` | `GameScore`の結果を受け取り、CSVファイルとして出力する。 |

## 🎮 試合の進行（操作方法）

プログラムを実行すると、以下の順序で入力が求められます。

1.  **リーグ/チーム選択:** 画面の指示に従い、ホームチームとアウェイチームを選択します。
2.  **ラインナップ設定:** 両チームの1番から9番までの打者名を順番に入力します。
3.  **先発投手設定:** 両チームの先発投手名を入力します。
4.  **イニング進行:** 1回表から始まり、3アウトになるまでシミュレーションが続きます。

### 打席ごとの入力コマンド

打席が始まると、一球ごとに以下の入力を求められます。

| コマンド | 入力例 | 意味 | 処理結果 |
| :--- | :--- | :--- | :--- |
| `B` | `b` | Ball | ボールカウントに加算 (4で四球) |
| `S` | `s` | Strike | ストライクカウントに加算 (3で三振) |
| `W` | `w` | Walk | 強制的に四球を宣言 |
| `O` | `o` | Out | 強制的にアウトを宣言 (犠飛、ゴロなど) |
| `I` | `i` | In Play (打球) | **次の結果入力に進みます** |

#### `I` (In Play) 後の結果入力

`I`を入力した場合、打球の最終結果を以下から選択します。

| コマンド | 意味 | 処理結果 |
| :--- | :--- | :--- |
| `1B` | Single (単打) | 走者が1塁進塁。打者は一塁へ。 |
| `2B` | Double (二塁打) | 走者が2塁進塁。打者は二塁へ。 |
| `3B` | Triple (三塁打) | 走者が3塁進塁。打者は三塁へ。 |
| `HR` | Home Run (本塁打) | 走者全員と打者が得点。 |
| `O` | Out | アウトカウントに加算。 |
| `E` | Error (失策) | エラーカウントに加算。走者は1塁進塁。 |
